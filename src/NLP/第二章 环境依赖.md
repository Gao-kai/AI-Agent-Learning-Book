# 第二章 环境依赖

## 创建conda环境

1. 创建虚拟环境 并指定python版本

```bash
conda create -n nlp-practice python=3.13
```

1. 激活虚拟环境

```bash
conda activate nlp-practice
conda deactivate
```

1. 安装所需依赖

- pytorch: 深度学习框架 主要用于模型的构建、训练和推理
- jieba: 中文分词库 用于将中文文本转换为词语序列
- gensim: 用于训练词向量模型 比如Word2Vec、FastText,提升模型对词语的理解能力
- transformers: Hugging Face提供的预训练模型库，用于加载预训练的BERT、GPT等主流模型
- datasets: Hugging Face提供的数据集库，用于加载和处理常用的NLP数据集
- TensorBoard: 用于可视化模型训练过程中的指标和损失函数
- tqdm: 用于显示训练进度条 提升训练效率
- Jupyter Notebook: 用于交互式开发和实验 方便调试和可视化

 ```bash
# 激活你的conda环境
conda activate nlp-practice

# 安装PyTorch（Apple Silicon版本）
# python -m pip 可以确保使用的是当前激活环境中的 Python 和 pip
python -m pip install torch torchvision torchaudio
python -m pip install jieba gensim transformers datasets tensorboard tqdm jupyter
```
