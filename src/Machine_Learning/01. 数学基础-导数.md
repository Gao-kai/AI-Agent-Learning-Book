## 1. 导数的基本概念

导数就是求函数在任意一点上的切线斜率。导数也叫做导函数值。

## 3. 导数的四则法则

![导数的求导法则](./image/01.数学基础-导数/1773936945905.png)

1. A加B求导，等于A导加B导
2. A乘B求导，等于A导B加上AB导
3. A除B求导，等于A导B减去B导A，然后除以B的平方
4. 复合函数的导数，需要先对外函数求导，再对内函数求导

## 4. 导数求极值

导数为0的点是函数的极值点。可能是极大值，也可能是极小值。
注意极大值和极小值不一定是最大值和最小值。

因此如果我们已经得到一个原函数（比如线性回归模型中的损失函数），现在要求这个函数的极小值：

1. 首先对原函数fx进行求导，得到导函数f‘x。
2. 基于导函数为0的点一定是原函数的极值点这一规则，让导函数f‘x等于0，求解x的值
3. 将x带入到原函数fx中得到y
4. y的值就是原函数的极大或极小值

## 5. 偏导数

### 什么是偏导数？

偏导数是**多元函数**对其中一个变量的导数，而将其他变量视为常数。

对于二元函数 $f(x, y)$：

- **对 x 的偏导数** $\frac{\partial f}{\partial x}$：把 y 当作常数，只对 x 求导
- **对 y 的偏导数** $\frac{\partial f}{\partial y}$：把 x 当作常数，只对 y 求导

### 符号表示

偏导数用希腊字母 **∂**（读作 "partial" 或 "偏"）表示：

| 表示方式                          | 含义                    |
| --------------------------------- | ----------------------- |
| $\frac{\partial f}{\partial x}$ | f 对 x 的偏导数         |
| $\frac{\partial f}{\partial y}$ | f 对 y 的偏导数         |
| $f_x$ 或 $f'_x$               | f 对 x 的偏导数（简写） |

### 示例

对于函数 $f(x, y) = x^2 + 2xy + y^2$：

**求 $\frac{\partial f}{\partial x}$（把 y 当常数）：**

$$
\frac{\partial f}{\partial x} = 2x + 2y
$$

**求 $\frac{\partial f}{\partial y}$（把 x 当常数）：**

$$
\frac{\partial f}{\partial y} = 2x + 2y
$$

### 为什么偏导数为 0 能找到极值？

想象你站在一个**碗状的山谷**里，想找到最低点：

```
        y
        ↑
        |     · · ·
        |   ·   ·   ·
        |  ·    ★    ·   ← 最低点
        |   ·   ·   ·
        |     · · ·
        └────────────→ x
```

假设碗的形状用函数表示：

$$
f(x, y) = x^2 + y^2
$$

这是一个开口向上的抛物面，最低点在 (0, 0)。

**求偏导数：**

对 x 求偏导（把 y 当常数）：$\frac{\partial f}{\partial x} = 2x$

对 y 求偏导（把 x 当常数）：$\frac{\partial f}{\partial y} = 2y$

**为什么让偏导数等于 0？**

| 方向   | 偏导数的含义      | 偏导数为 0 的意义      |
| ------ | ----------------- | ---------------------- |
| x 方向 | 沿 x 轴方向的坡度 | x 方向走不动了（平的） |
| y 方向 | 沿 y 轴方向的坡度 | y 方向走不动了（平的） |

**直观理解：**

- 偏导数 = 坡度（斜率）
- 坡度 ≠ 0 → 还在下坡，不是最低点
- 坡度 = 0 → 这个方向"平"了，可能是最低点

**解方程组：**

$$
\begin{cases} \frac{\partial f}{\partial x} = 2x = 0 \\ \frac{\partial f}{\partial y} = 2y = 0 \end{cases}
$$

解得：**x = 0, y = 0**，这就是最低点！

**一句话总结：** 偏导数为 0 = 在该方向上"站平了"，所有方向都"站平了"，就是极值点。

### 偏导数的作用

| 应用场景                   | 说明                                       |
| -------------------------- | ------------------------------------------ |
| **求多元函数极值**   | 让所有偏导数等于 0，解方程组               |
| **梯度计算**         | 偏导数组成梯度向量，指示函数变化最快的方向 |
| **机器学习优化**     | 梯度下降中计算损失函数对各参数的偏导数     |
| **神经网络反向传播** | 链式法则计算偏导数更新权重                 |

### 与线性回归的关系

在损失函数 `Loss(k, b)` 中，k 和 b 都是变量：

- $\frac{\partial Loss}{\partial k} = 0$ → 求出最优 k
- $\frac{\partial Loss}{\partial b} = 0$ → 求出最优 b

联立两个方程，就能同时找到让损失函数最小的 k 和 b 值
