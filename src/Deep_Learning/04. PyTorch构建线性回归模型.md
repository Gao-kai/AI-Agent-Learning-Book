# 基于PyTorch构建线性回归模型

使用PyTorch框架提供的各个组件来构建线性回归模型，需要分为四个步骤:

1. 准备训练集数据
2. 构建要使用的模型
3. 设置损失函数
4. 设置优化器
5. 模型训练

我们使用到的API为:

1. nn.MSELoss() 实现均方误差损失函数
2. data.DataLoader() 实现数据加载器
3. optim.SGD() 实现随机梯度下降优化器
4. nn.Linear() 实现线性回归模型

numpy -> tensor -> 数据集对象Dataset
数据集对象Dataset -> 数据加载器Dataloader
数据加载器Dataloader -> 模型Model
