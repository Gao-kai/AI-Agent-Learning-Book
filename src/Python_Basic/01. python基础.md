## 第二节 编码格式

首先我们需要了解下编码的发展历史，并由此进一步解释Unicode字符集、utf-8编码等相关概念

- 最早编码为ASCII编码，支持大部分大写字母、小写字母、数字和一些符号，总计128个字符
- ASCII无法表达希腊字母等特殊字符，因此欧洲推出了ISO 8859-1编码方法，在ASCII的基础上新增了128个字符，总计256个字符
- ASCII也无法表达中文和繁体中文字符等，因此中国依次推出了GB2312和GBK编码方法，GB2312支持6763个中文字符，GBK支持21003个中文字符

因此我们发现随着不同国家和地区使用的文字不同，会出现越来越多的字符编码，因此Unicode字符集就应运而生了，请注意Unicode是字符集，类似计算机世界的一本字典，它只用来定义世界上所有的字符在计算机世界中所对应的码点（码点是一个整数），而不是具体的二进制编码。

我们可以用代码:

- 使用ord方法查看任意一个字符的码点
- 使用encode方法将其转换为utf-8编码
- 通过decode方法将utf-8编码转换为字符

```py
text = '中国';

# 获取Unicode码点 也就是一个整数
print(ord(text[0])) # 20013
print(ord(text[1])) # 22269

# 获取码点对应的UTF-8编码 在计算机中存储的二进制形式
print(text[0].encode('utf-8')) # b'\xe4\xb8\xad'
print(text[1].encode('utf-8')) # b'\xe5\x9b\xbd'

# 编码encode
encoded = text.encode('utf-8')
print(encoded) # b'\xe4\xb8\xad\xe5\x9b\xbd'

# 解码decode
decoded = encoded.decode('utf-8')
print(decoded) # 中国
```

总结一句话：，也可以说utf-8是Unicode的一种实现方式。

最后还需要注意的编码方式utf-8、utf-16、utf-32的不同。

首先我们来了解下utf-8、utf-16、utf-32的不同。

- utf-8是一种变长编码，每个字符占用1-4个字节，根据字符的不同而变化。这里需要说明的是：
  - utf-8编码中，每个字符占用的字节数是根据字符的Unicode码点来确定的。
  - 码点在0-127之间的字符，比如ASCII字符，占用1个字节
  - 码点在128-2047之间的字符，比如拓展的拉丁字符，占用2个字节
  - 码点在2048-65535之间的字符，比如大多数中文，占用3个字节
  - 码点在65536-1114111之间的字符，比如生僻字等，占用4个字节
- utf-16是一种固定长度编码，每个字符占用2个字节。
- utf-32是一种固定长度编码，每个字符占用4个字节。

总结来说：

- Unicode是字符集，utf-8是一种编码格式
- Unicode可以理解为是一本字典，而utf-8是一本具体的编码实现，规定了如何检索和存储字符。
- Unicode的码点决定了一个字符在utf-8中存储几个字节，可能是1-4中的一种。
- utf-16和utf-32都是固定长度编码，每个字符占用2个字节和4个字节，也是编码方式。

最后有两个原则：

1. 存储时，必须采用合适的字符编码，否则会无法存储，比如py文件中的中文字符串如果在IDE中选择ISO 8859-1编码，就会出现原先的中文字符乱码的情况，导致后续无法正常解码。
2. 读取时，必须采用与存储时相同的字符编码，否则会出现解码错误。比如汉字在IDE中使用utf-8编码存储，但是在读取时通过 #coding=iso8859-1 告诉python解释器使用iso8859-1编码进行解码，就会出现解码错误。

## 第三节 数据类型

1. python中数据类型分为两大类：
   - 不可变数据类型
     - 数字 分为整数、浮点数，分别用int 和 float表示
     - 字符串 用str表示
     - 元组 用tuple表示
   - 可变数据类型
     - 列表 用list表示
     - 字典 用dict表示
     - 集合 用set表示

2. python中获取数据类型的方法有两种：

     ```py
     str = '你好啊'

     ```

   - 使用type()函数 获取数据类型的类名
   - 使用isinstance()函数 判断数据是否属于某个类型
   - 使用dir()函数 获取对象的所有属性和方法

3. Python是动态强类型语言，而javascript是动态弱类型语言。
   - 强类型语言意味所有类型转化都必须是显式的，而弱类型语言则允许隐式类型转换。
   - 动态类型语言意味变量的类型是在运行时确定的，而不是在编译时确定的。也就是变量的类型可以在程序运行过程中改变。

### 占位符精度控制详解

1. %m.nf
   - m 表示最小宽度，即占位符最少占用的字符数
      - 大于浮点数长度时，用空格填充
      - 小于浮点数长度时，占位符占用的字符数为浮点数长度。
   - n 表示精度，即保留n位小数，默认值为6
     - n大于浮点数的长度时用0补充
     - n小于浮点数的长度时，进行四舍五入截断处理
   - 正数表示右对齐，负数表示左对齐

```py
float_number = 65.56;

print('浮点数 ---%6.2f---' % float_number)
print('浮点数 ---%4.2f---' % float_number)
print('浮点数 ---%10.2f---' % float_number)
print('浮点数 ---%-10.2f---' % float_number)
print('浮点数 ---%6.4f---' % float_number)
print('浮点数 ---%-6.1f---' % float_number)


# 1. 先读取n，如果n的值小于浮点数的长度，进行四舍五入；否则n是多少最终小数部分就是几位。
# 2. 再读取m，如果m的值大于浮点数的长度，用空格填充；否则m是多少最终整个浮点数就是几位。

# 依次输出为
# 浮点数 --- 65.56---
# 浮点数 ---65.56---
# 浮点数 ---     65.56---
# 浮点数 ---65.56     ---
# 浮点数 ---65.5600---
# 浮点数 ---65.6  ---
```

## 第四节 运算符

### 算术运算符

和javascript中的算术运算符基本一致，下面是几个不同点：

1. 取整运算符 // 表示整除，返回两个操作数相除的整数部分，不保留小数部分。
2. 取模运算符 % 表示取余运算，返回两个操作数相除的余数部分。
3. 除法运算符 / 表示普通除法，返回两个操作数相除的结果，保留小数部分。js中如果整除不会保留小数部分。

### 比较运算符

和javascript中的比较运算符基本一致，下面是几个不同点：

1. 相等==比较运算符必须要求两侧的值和数据类型完全一致，类似js中的全等比较运算符===
2. 不相等!=比较运算符必须要求两侧的值和数据类型完全一致，类似js中的不全等比较运算符!==
3. 大于>和小于<比较等运算符要求两侧的值的数据类型必须完全一致，也不能将一个字符串和一个数值进行比较。
   但是这在js中是可以的，根本原因是js是弱类型语言，可以自动进行类型的隐式转换。
   但是python是强类型语言，不可以将一个字符串和一个数值进行比较。

### python语言中字符串比较的内部机制

1. 从头到尾按照顺序比较每一个字符的Unicode编码值。
2. 如果两个字符串的第一个字符的Unicode编码值不同，那么就可以直接判断出这两个字符串的大小关系。
3. 如果两个字符串的第一个字符的Unicode编码值相同，那么就需要比较第二个字符的Unicode编码值，以此类推。
4. 如果前缀一致，比较长度即可。

常见的几个内置方法：

1. len() 可以获取字符串的长度
2. ord() 可以将一个字符转换为对应的Unicode编码值
3. chr() 可以将一个Unicode编码值转换为对应的字符
4. 字符串的比较是按照Unicode编码值进行比较的

### 逻辑运算符

1. and（类似js中的&&） 逻辑与运算符 要求两侧的操作数都为True，才会返回True，否则返回False
2. or（类似js中的||） 逻辑或运算符 要求两侧的操作数有一个为True，就会返回True，否则返回False
3. not（类似js中的!） 逻辑非运算符 要求操作数为True，才会返回False，否则返回True

需要注意的几个点是：

1. and和or依然具备逻辑短路的特性
2. and和or运算符并不一定返回的都是布尔值，它返回的是参与运算的值本身，举例说明：
   print(2 - 2 and True)这行代码在执行的时候：
   - 先计算2-2的值，此时python会将不是布尔值的其他类型的值都转化为布尔类型，这里计算为0转化为False
   - 左边是False，因此右边短路不执行
   - 最终返回计算的结果0，而不是布尔值False，布尔值仅仅是在底层运算的时候用来判断逻辑的，而不是最终的结果。
3. not运算符返回的一定是布尔值

## 第五节 进制

### 将任意进制的值转化为10进制的过程

比如二进制的11001转化为10进制本质就是在做（幂次方的加法）

1. 从右往左依次计算
2. 将每个位上的数字乘以2的幂次方，
3. 将所有的结果相加，得到最终的10进制值

例如：1*2^0 + 0*2^1 + 0*2^2 + 1*2^3 + 1*2^4 = 1 + 0 + 0 + 8 + 16 = 25

### 将10进制转化为任意进制的过程

本质就是连除法直到商为0

1. 连续进行除法运算，从上到下记录余数
2. 当最终商为0的时候
3. 最后将所有的余数倒序排列起来，得到最终的结果

例如：将10进制的25转化为二进制

1. 25 ÷ 2 = 12 余 1
2. 12 ÷ 2 = 6  余 0
3. 6 ÷ 2 = 3   余 0
4. 3 ÷ 2 = 1   余 1
5. 1 ÷ 2 = 0   余 1 （STOP）
6. 最后将所有的余数倒序排列起来，得到最终的结果：11001

## 第六节 语句

1. input() 语句可以从用户输入中获取一个字符串，获取到的值一定为字符串.
